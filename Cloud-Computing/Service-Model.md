# ☁️ Cloud Computing Service Models

## 📖 Introduction

Cloud service models define **how cloud services are delivered to users**. They determine **who manages what** in the cloud environment.

The three primary cloud service models are:

- 🖥️ Infrastructure as a Service (IaaS)
- ⚙️ Platform as a Service (PaaS)
- 💻 Software as a Service (SaaS)

---

# 🏗️ 1. Infrastructure as a Service (IaaS)

## 📖 Definition

Infrastructure as a Service (IaaS) provides **virtualized computing resources** such as servers, storage, networking, and virtual machines over the internet.

The cloud provider manages the physical infrastructure, while the customer manages the operating system, applications, and data.

## 👨‍💼 Responsibilities

### Cloud Provider

- Physical Servers, Networking, Storage,Virtualization


### Customer

- Operating System,  Middleware,  Runtime,  Applications,  Data

### 🌍 Examples

- Amazon EC2, Microsoft Azure Virtual Machines, Amazon EBS

### 🎯 Use Cases

- Website hosting, Development environments,  Disaster recovery,  Big data processing,  Backup solutions

---

# ⚙️ 2. Platform as a Service (PaaS)

## 📖 Definition

Platform as a Service (PaaS) provides a complete platform for developing, testing, deploying, and managing applications without managing servers.

The cloud provider manages the infrastructure and platform.


## 👨‍💼 Responsibilities

### Cloud Provider

- Infrastructure,  Operating System,  Runtime,  Middleware,  Database

### Customer

- Applications,  Data


### 🌍 Examples

- AWS Elastic Beanstalk, Azure App Service,  Google App Engine

### 🎯 Use Cases

- Web application development, Mobile application backend,  API development,  Testing environments

---

# 💻 3. Software as a Service (SaaS)

## 📖 Definition

Software as a Service (SaaS) delivers complete software applications over the internet.

Users access the software through a web browser without installing or maintaining it.


## 👨‍💼 Responsibilities

### Cloud Provider

- Infrastructure, Platform,  Software,  Security,  Maintenance,  Updates

### Customer

- Uses the application


### 🌍 Examples

- Gmail, Microsoft 365,  Google Workspace, Zoom, Dropbox 


### 🎯 Use Cases

- Email services, Video conferencing, CRM software,   Cloud storage
---

# 🆚 Quick Comparison

| IaaS | PaaS | SaaS |
|------|------|------|
| Virtual Infrastructure | Development Platform | Ready-to-use Software |
| Manage OS | No OS Management | No Infrastructure Management |
| Maximum Control | Medium Control | Minimum Control |
| Flexible | Fast Development | Easy to Use |
