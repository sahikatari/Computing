# DEPLOYMENT MODEL

Cloud computing can be categorized based on how cloud services are deployed and managed. The deployment models are:

- 🌍 Public Cloud
- 🔒 Private Cloud
- 🔄 Hybrid Cloud
- 🤝 Community Cloud
- ☁️ Multi-Cloud


---

# 🌍 1. Public Cloud

The public cloud delivers infrastructure and services over the internet to the general public.
A Public Cloud is owned and managed by a third-party cloud service provider.
Resources such as servers, storage, and networking are shared among multiple customers over the internet.

## Examples

- Amazon Web Services (AWS)
- Microsoft Azure
- Google Cloud Platform (GCP)

---

# 🔒 2. Private Cloud

A Private Cloud is dedicated to a single organization.
It can be hosted on-premises or by a third-party provider.
It is also know as **internal cloud**.

## Examples

- VMware Private Cloud
- OpenStack
- Azure Stack

---

# 🔄 3. Hybrid Cloud

A Hybrid Cloud combines Public Cloud and Private Cloud, allowing data and applications to move between both environments.

## Examples

- AWS + On-Premises Data Center
- Azure + Private Cloud
- Google Cloud + Local Servers

---

# 🤝 4. Community Cloud

A Community Cloud is shared by multiple organizations that have similar security, compliance, or business requirements.
Best for organizations working in the same industry with shared compliance requirements.

## Features

- Shared infrastructure
- Cost shared among organizations
- High security
- Common policies and regulations

## Examples

- Government departments
- Hospitals
- Universities
- Research organizations

---

  # ☁️ 5. Multi-Cloud

A Multi-Cloud strategy uses services from multiple cloud providers instead of relying on a single provider.

## Features

- Uses multiple cloud vendors
- Avoids vendor lock-in
- Improves reliability
- Optimizes cost and performance

## Example

- AWS for computing (EC2)
- Azure for Active Directory
- Google Cloud for AI and Machine Learning

---


# 📝 Key Takeaways

- 🌍 Public Cloud is affordable and highly scalable.
- 🔒 Private Cloud provides maximum security and control.
- 🔄 Hybrid Cloud combines the benefits of Public and Private Clouds.
- 🤝 Community Cloud is shared by organizations with common goals and compliance needs.
- ☁️ Multi-Cloud uses multiple cloud providers to improve flexibility, reliability, and avoid vendor lock-in.
